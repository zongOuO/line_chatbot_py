# 使用Line Chatbot聊天室跟LLM進行連續對話
指令：  
1.!清空對話紀錄：會清除先前的聊天紀錄  
2.!預設LLM回覆語氣：回復預設LLM回覆語氣  
3.!修改LLM回覆語氣+提示詞：可以修改LLM回覆的語氣  
4.提到天氣查詢、查詢天氣、(查詢&天氣)+城市：會提供該城市的天氣預報
